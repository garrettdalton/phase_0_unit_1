Why Pair Program?

To Garrett and myself, programming is essential because it reflects skills needed in the real world and because it's an excellent way to learn the material. Modern software is created by teams of people, not one person sitting in solitude. It's a social experience, thus it is vital to get practice in learning with others. In addition, programming in pairs allows for pairs to bounce questions and ideas off of each other. This exercise makes learning concepts easier as it provides an opportunity to gain a second perspective.
